第1周周报

一、项目实践内容
1. 前端开发环境搭建
本周是项目启动的第一周，主要完成了客户端开发环境的配置工作。安装了 Visual Studio Code 编辑器，并配置了相关的开发插件以提高编码效率。同时，安装了 Node.js 运行环境，为后续的前端工具链支持做准备。
此外，完成了 Git 版本控制工具的安装与配置，通过 SSH 密钥与远程代码仓库建立了连接，初步掌握了代码拉取和提交的流程。

2. 基础代码实践
创建了项目的基础 HTML 文件，编写了标题、段落、列表等基础标签，熟悉了 HTML5 的文档结构。
创建了 CSS 样式文件，练习了字体颜色、背景色等基础样式的设置，并成功将其引入到 HTML 页面中，实现了结构与样式的分离。
初步尝试了 JavaScript 代码的编写，通过浏览器控制台输出日志，验证了脚本的正常运行。

二、实践收获
1. 熟悉了现代前端开发的常用工具链，特别是 VS Code 编辑器的使用，其丰富的插件生态对提升开发效率有很大帮助。
2. 明确了网页开发的三大核心技术分工：HTML 负责页面结构，CSS 负责页面表现，JavaScript 负责页面交互。
3. 掌握了基本的 Git 版本控制操作，理解了代码提交（commit）和推送（push）的基本概念。

三、实践体会
在环境搭建过程中遇到了一些路径配置的问题，通过查阅相关文档和资料最终得以解决。这一过程让我认识到，开发环境的正确配置是项目顺利开展的基础。虽然目前编写的代码还比较简单，但看到页面能够正常显示，对后续的开发工作充满了信心。

四、建议
在配置开发环境时，建议仔细阅读官方文档，注意操作系统环境变量的设置。对于 Git 命令的使用，建议整理一份常用命令清单，以便在日常开发中快速查阅，避免操作失误。
