第3周周报

一、项目实践内容
1. CSS 盒模型学习
本周深入学习了 CSS 布局的核心——盒模型。通过编写不同边框和内边距的测试案例，分析了元素宽高的计算方式。使用了 `box-sizing: border-box` 属性，解决了布局中常见的尺寸计算问题。

2. Flexbox 弹性布局实践
针对传统布局方式的局限性，学习了 Flexbox 弹性布局。通过设置 `display: flex` 以及 `justify-content`、`align-items` 等属性，实现了元素的水平垂直居中排列，大大简化了布局代码。

3. 基础响应式设计
尝试使用了 CSS 媒体查询（Media Queries），根据不同的屏幕宽度应用不同的样式规则。实现了简单的响应式效果，使页面在移动端和桌面端能够呈现不同的布局结构。

二、实践收获
1. 深入理解了盒模型的组成（Margin, Border, Padding, Content），掌握了标准盒模型与怪异盒模型的区别及其应用场景。
2. 掌握了 Flexbox 布局的基本用法，体会到了其在处理一维布局时的灵活性和便捷性。
3. 了解了响应式设计的基本原理，认识到网页布局应当适应不同尺寸的终端设备。

三、实践体会
CSS 样式的调试需要极大的耐心，尤其是在处理元素对齐和间距时。通过本周的练习，我发现良好的布局习惯（如优先使用 Flexbox）可以减少很多不必要的样式调整工作。同时，理解浏览器默认样式对布局的影响也十分重要。

四、建议
当遇到样式不生效或布局错乱的问题时，应充分利用浏览器的开发者工具（DevTools），查看元素的计算样式（Computed）和盒模型示意图，这能快速定位问题所在。
