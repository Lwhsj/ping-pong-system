第4周周报

一、项目实践内容
1. JavaScript 基础语法练习
本周开始系统学习 JavaScript 编程语言。重点练习了变量的声明方式，逐渐使用 `let` 和 `const` 替代了传统的 `var`。尝试编写了箭头函数，简化了函数定义的语法结构。

2. 模板字符串应用
在处理字符串拼接时，使用了 ES6 引入的模板字符串（Template Strings）。通过反引号和 `${}` 语法，实现了变量与字符串的便捷拼接，提高了代码的可读性。

3. 基础逻辑控制
编写了简单的逻辑判断函数，练习了 `if-else` 条件语句和 `switch` 分支语句的使用。同时，初步尝试了数组的解构赋值，简化了数据提取的操作。

二、实践收获
1. 理解了 `let`、`const` 与 `var` 在作用域上的区别，养成了良好的变量声明习惯。
2. 认识到 JavaScript 的异步执行特性，虽然理解尚浅，但已了解到代码执行顺序并非完全线性的。
3. 掌握了模板字符串的使用方法，有效避免了传统字符串拼接中容易出现的符号错误。

三、实践体会
相比于 HTML 和 CSS，JavaScript 的逻辑性更强，入门难度也稍大。在理解作用域、闭包等抽象概念时遇到了一定挑战。此外，JavaScript 的错误提示有时不够直观，需要花费更多时间进行排查。

四、建议
学习编程语言最重要的是动手实践。建议多编写一些小型的功能 Demo，如简单的计算器或逻辑判断程序，通过实际运行代码来加深对语法的理解，而不是仅停留在阅读文档层面。
