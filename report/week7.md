第7周周报

一、项目实践内容
1. Vue.js 框架入门
本周正式开始了 Vue.js 框架的学习。通过实例化 Vue 对象，将之前的“待办事项”功能进行了重构。利用 Vue 的声明式渲染特性，通过简单的数据定义和循环指令，实现了列表的自动渲染，大幅减少了手动的 DOM 操作。

2. 双向数据绑定实践
重点练习了 `v-model` 指令，实现了表单输入框与数据状态的双向绑定。体验了数据变化自动驱动视图更新的开发模式，无需再手动获取和设置输入框的值。

3. 常用指令应用
系统练习了 Vue 的常用指令，包括用于条件渲染的 `v-if` 和 `v-show`，用于属性绑定的 `v-bind`，以及用于事件处理的 `v-on`。

二、实践收获
1. 深刻体会到了“数据驱动视图”的设计理念。开发者只需关注业务数据的逻辑，页面的更新交由框架自动处理，极大地简化了开发流程。
2. 掌握了 Vue 模板语法的基本使用，逻辑代码与视图模板的分离使得代码结构更加清晰。
3. 辨析了 `v-if`（条件渲染）和 `v-show`（样式切换）的区别，并了解了它们各自适用的场景。

三、实践体会
从原生 JavaScript 切换到 Vue 框架，开发思维发生了转变。虽然需要学习新的语法规则，但开发效率得到了质的飞跃。初期容易遗漏在 data 选项中声明变量，导致报错，需要通过不断的练习来适应框架的规范。

四、建议
在处理复杂逻辑时，建议优先使用 `computed` 计算属性，而不是在模板中编写复杂的表达式。这样可以保持模板的简洁性，提高代码的可维护性。
