第2周周报

一、项目实践内容
1. 页面结构优化
本周重点进行了 HTML 页面结构的练习。在制作个人简介页面时，尝试将原有的 `div` 布局替换为更具语义化的标签。使用了 `<header>`、`<footer>`、`<nav>` 等 HTML5 新标签来构建页面框架，使代码结构更加清晰。

2. 表单功能实现
制作了一个基础的信息收集表单，练习了 `<input>`（文本框、邮箱）、`<textarea>`（文本域）等表单元素的使用。同时也尝试了表单的提交操作，观察了数据提交时的页面刷新现象。

二、实践收获
1. 理解了“HTML 语义化”的概念，即使用正确的标签来描述内容。这不仅有助于代码的可读性，也有利于搜索引擎的抓取和辅助设备的识别。
2. 区分了行内元素（如 `span`）和块级元素（如 `div`）的显示特性，了解了它们在布局中的不同表现。
3. 掌握了表单元素的基本属性设置，如使用 `placeholder` 设置提示文字，使用 `required` 进行非空验证。

三、实践体会
HTML 标签种类繁多，初期记忆较为困难，但通过实际编码练习，逐渐掌握了常用标签的用法。使用语义化标签后，代码的可读性有了明显提升，不再是满屏的 `div`，这对于代码的后期维护非常重要。

四、建议
在编写 HTML 结构前，建议先绘制简单的草图，规划好页面的区域划分，确定头部、主体和底部的内容。这样在编写代码时思路会更加清晰，能够有效减少结构混乱的情况。
