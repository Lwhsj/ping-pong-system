第9周周报

一、项目实践内容
1. Vue Router 路由配置
为了构建单页应用（SPA），本周引入了 Vue Router 进行路由管理。配置了主页、关于页等基础路由规则，并使用 `<router-link>` 组件实现了页面的无刷新跳转，提升了用户体验。

2. 路由守卫应用
尝试实现了全局前置守卫 `beforeEach`。模拟了简单的权限控制场景：检查本地存储中是否存在用户凭证（Token），若不存在则强制跳转至登录页面。在调试过程中解决了因逻辑错误导致的无限重定向问题。

3. 生命周期钩子实践
通过编写测试代码，观察了 Vue 组件的生命周期。在 `onMounted` 钩子中进行数据请求模拟，在 `onUnmounted` 钩子中进行定时器清理，加深了对组件创建、挂载、销毁过程的理解。

二、实践收获
1. 理解了单页应用的路由原理，Vue Router 通过 URL 的变化动态切换组件，而无需重新加载整个页面。
2. 掌握了生命周期的各个阶段及其应用场景，明确了在何处进行 DOM 操作或资源清理最为合适。
3. 认识到路由守卫在应用安全和权限管理方面的重要作用，它是保护特定页面资源的有效手段。

三、实践体会
生命周期的概念初期较为抽象，尤其是父子组件的生命周期执行顺序容易混淆。通过实际打印日志观察执行顺序，才真正理清了组件的加载流程。这对于解决数据加载时机的问题非常有帮助。

四、建议
在编写路由跳转逻辑时，需特别注意避免死循环的情况。此外，避免在生命周期钩子中执行过重的同步操作，以免阻塞页面渲染，影响用户体验。
